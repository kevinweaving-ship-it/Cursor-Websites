Cape Classic Series (test)

This page is a test view only. It shows how a series can be built from three separate regatta results already in the results table — it is not a fourth regatta and not the same URL pattern as normal regatta results.

Each leg stays a normal regatta (search / regatta pages unchanged). Here we only aggregate by helm SA id across three Cape Classic legs for a given class.

The three legs (same season snapshot)

ZVYC — 359-2025-zvyc-southern-charter-cape-classic

TSC — 371-2025-tsc-cape-classic

HYC — 380-2026-hyc-cape-classic (blocks may use 385-2026-hyc-cape-classic:*)

Open each regatta’s results as usual; this series page only groups rows that share the mapped block_ids per class below.

How the series table is worked out

  • Eligibility: helm must have a result in all three legs for that class (3 of 3), where that class actually runs at each venue. If a venue does not host the class (e.g. Hobie at ZVYC — water too shallow), the fair comparison is over the available legs only; sailors who sailed those legs but not a missing class leg are listed as “2 of 3” (or “2 of 2” when only two venues have that fleet).
  • Partial series: helms with only two legs when three exist for that class → listed separately as “2 of 3”, not on the main series podium.
  • Ranking: lowest sum of leg ranks; tie-break: lowest sum of leg netts.
  • Grouping: GROUP BY helm_sa_sailing_id (display names can change between events).

Other classes — venue limits and “2 of 3”

Some fleets are not sailed at every Cape Classic host: for example Hobie catamarans are not raced at ZVYC in this series snapshot (draft / water depth), so a Hobie “series” here is only TSC + HYC — two legs, not three. Mirror has ZVYC and TSC blocks in the data; there is no Mirror block at HYC in this snapshot, so the comparable series is ZVYC + TSC. Use the same rank-sum / nett-sum rules on those available legs.

ILCA 4.7 and 420 below use full 3-of-3 where all three venues hosted that class.
