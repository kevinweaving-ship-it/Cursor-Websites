Landing page artwork (Sigma / Figma redesign)
==============================================

Purpose
-------
Drop logos, icons, and exported artwork here so they stay in the repo and can be handed to Cursor or developers later. This folder is NOT under `assets/` (which is gitignored) — commit these files so they are backed up.

Suggested use
-------------
• logos/     — Wordmarks, flag marks, full Sailing SA logo, SVG/PNG masters
• icons/     — UI icons, pictograms for the new landing (PNG/SVG)
• exports/   — PNG/JPG/SVG slices from Sigma or Figma (hero, sections, etc.)
• reference/ — Screenshots, PDFs, mood boards (optional)

Live site
---------
Production still serves static files from `/assets/...` on the server. When a design is final, copy chosen assets into `sailingsa/frontend/assets/` (or the paths the design system uses) and deploy.

Naming
------
Use clear names: e.g. `logo-wordmark-dark.svg`, `hero-illustration-2026.png`
