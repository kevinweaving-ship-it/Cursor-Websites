# ONE COMMAND SETUP

## Run This:

```bash
cd sailingsa/backend
npm run setup-google
```

This will:
1. Open Google Cloud Console
2. Show you exactly what to do
3. Let you paste your credentials
4. Save them automatically
5. Restart the backend

## That's It.

After running `npm run setup-google`, just follow the prompts and paste your Google OAuth credentials when asked.
