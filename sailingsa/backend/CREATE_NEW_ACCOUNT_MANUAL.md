# Create New Google Account for sailingsa.co.za - MANUAL ONLY

## ⚠️ CRITICAL: NO AUTOMATION

The previous account was disabled because Google detected automation.
You MUST create this account 100% manually - no scripts, no automation.

## Steps (Do Everything Yourself):

1. **Page opened**: Google signup page should be open
   - If not, go to: https://accounts.google.com/signup

2. **Fill form manually** (type everything yourself):
   - First name: **SailingSA**
   - Last name: **Admin**
   - Username: Try one of these (type manually):
     - `sailingsaadmin@gmail.com`
     - `sailingsaweb@gmail.com`
     - `sailingsa.co.za@gmail.com`
     - `sailingsaresults@gmail.com`
   - Password: **Type a strong password manually**
     - Mix of letters, numbers, symbols
     - Save it securely!

3. **Complete CAPTCHA manually**:
   - Click/tap the CAPTCHA yourself
   - Complete any verification yourself

4. **Verify email manually**:
   - Check the email inbox for the username you chose
   - Click verification link yourself

5. **Complete all remaining steps manually**:
   - Add phone (if required)
   - Accept terms
   - Finish setup

## After Account Created:

1. **Login to Google Cloud Console** with new account
2. **Run OAuth setup**:
   ```bash
   cd sailingsa/backend
   npm run setup-google
   ```

## Why Manual?

- Google's security detects ANY automation
- Automated accounts get disabled immediately
- Manual creation = account stays active
- Takes 2 minutes manually
