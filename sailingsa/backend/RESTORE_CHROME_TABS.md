# Restore Chrome Tabs

## Quick Methods:

### Method 1: Reopen Closed Tabs (Fastest)
**Press:** `Cmd + Shift + T`
- Reopens the most recently closed tab
- Press multiple times to reopen more tabs
- Works for tabs closed in current session

### Method 2: Chrome History
**Press:** `Cmd + Y` (or Chrome menu → History)
- I've opened this for you
- Browse your history
- Click any page to reopen it
- Right-click → "Open in new tab" for multiple

### Method 3: Recently Closed Windows
1. Chrome menu → **History** → **Recently Closed**
2. Click "X tabs" to restore entire window
3. Or click individual tabs

### Method 4: Session Restore
If Chrome crashed:
- Chrome should ask to restore on next launch
- Or: Chrome menu → **History** → **Restore tabs**

## What I Did:

1. ✅ Opened Chrome History (Cmd+Y)
2. ✅ Attempted to restore last closed tab (Cmd+Shift+T)

## If Tabs Still Missing:

- Check **History** (Cmd+Y) - all visited pages are there
- Check **Bookmarks** if you had any bookmarked
- Check **Other devices** if you sync Chrome:
  - Chrome menu → **History** → **Tabs from other devices**

## Prevention:

- Enable Chrome sync to save tabs across devices
- Use bookmarks for important pages
- Chrome automatically saves session on close
